INTRO WITH PROJECT

            Project นี้จัดทำขึ้นเพื่อวิเคราะห์ตรวจสอบและค้นหาเกมส์และ
เหล่าสตรีมเมอร์ชื่อดังทั่วโลกหลากหลายประเทศที่คนนิยมเข้าชม
โดยอ้างอิงจาก API ของ Twitch และนำมา วิเคราะห์ด้วยโปรแกรมต่างๆ แล้วจึงนำมาสร้างกราฟ เพื่อเปรียบเทียบข้อมูลในแต่ละส่วน
